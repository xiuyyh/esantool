# **App Name**: Esan Tools

## Core Features:

- Product Listing Management: Admin can create, update, and manage detailed listings for Telegram group links, social media logs, VPN logs, and custom website development services, including pricing, categories, and descriptions.
- Secure Buyer/Admin Authentication: Robust user authentication and authorization system to differentiate between buyers and admin and secure access to their respective functionalities.
- Product Browsing & Search: Buyers can browse available digital products and services through categorized listings and use a search function to find specific items.
- Integrated Payment Processing: Integration with a secure payment gateway to handle transactions for digital goods and services efficiently and safely.
- Order Management & Delivery: System for admin to track sales and deliver digital products, and for buyers to securely access purchased items and services.
- Admin Dashboard: A dedicated dashboard for admin to monitor product listings, view sales performance, and manage orders.

## Style Guidelines:

- A dark color scheme evokes a sense of digital sophistication and security. The primary color, a deep blue-violet ("#2E38AF"), provides a strong, technological foundation. The background is a very dark, subtly blue-tinted grey ("#15171C"), allowing content to stand out. A vibrant light blue accent ("#32C4F0") highlights interactive elements and draws attention to key information, creating visual contrast and modernity.
- Headlines use 'Space Grotesk' (sans-serif) for a modern, tech-inspired aesthetic, suitable for short, impactful text. Body text utilizes 'Inter' (sans-serif) for its neutrality, readability, and clean appearance, ensuring clear communication of product details and user interface elements.
- Employ a set of sleek, minimalist line-art vector icons. These icons will maintain a modern, digital aesthetic and provide clear, intuitive visual cues without distracting from the content, reinforcing the platform's professional feel.
- The layout is clean and structured, utilizing a grid-based system for product listings to ensure consistency and readability. Ample use of negative space provides a sophisticated, uncluttered user experience, directing focus to the content and interactions. Components are clearly defined with subtle borders or shadows.
- Subtle, smooth hover effects on interactive elements such as buttons and product cards, and seamless transitions for page changes or data loading. These animations enhance user feedback and contribute to a premium, fluid browsing and interaction experience without being overly distracting.
